# Artistic-Robot

## Contribute 

### Clone
`git clone https://github.com/mennatawfiq/Artistic-Robot`

### Activate venv
```
source .venv/bin/activate   # Linux/mac
# or
.venv\Scripts\activate      # Windows
```

### Install packages
```
pip install -r requirements.txt

```

### Run
```
python main.py
```

### After each `pip install`
run
```
pip freeze > requirements.txt
```
